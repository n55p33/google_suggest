google suggestで単語候補を表示する秀丸マクロ。

＊インストール
    下記2ファイルを秀丸エディタのマクロディレクトリにコピーしてください。
        google_suggest.mac
        google_suggest_sub.js

    最後に、google_suggest.macをキーアサインすればインストールは完了です。
    （キーアサインは必須ではありません。）
